Implementation of better-sqlite3-based express-session-compatible session store:
https://expressjs.com/en/resources/middleware/session.html
https://github.com/WiseLibs/better-sqlite3/blob/master/docs/api.md

Intended for use in personal projects, public because my portfolio is lacking, and completely free to use however desired by anyone. I'd recommend you implement your own solution if you, like me, are looking for an opportunity to try more stubbing & mocking.
